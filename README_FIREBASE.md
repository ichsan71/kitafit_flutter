## 🔑 Key Points

### Firebase vs Supabase

| Aspek           | Supabase           | Firebase                                            |
| --------------- | ------------------ | --------------------------------------------------- |
| **Auth**        | Supabase Auth      | Firebase Auth                                       |
| **Database**    | PostgreSQL         | Firestore                                           |
| **Real-time**   | Via WebSocket      | Built-in                                            |
| **Offline**     | Limited            | Excellent                                           |
| **SDK**         | `supabase_flutter` | `firebase_core`, `firebase_auth`, `cloud_firestore` |
| **File Upload** | Storage API        | Firebase Storage                                    |

### Architecture Changes

```
BEFORE (Supabase):
SupabaseClient
  ├─ auth.signInWithPassword()
  ├─ auth.signUp()
  └─ from('profiles').select()

AFTER (Firebase):
FirebaseAuth
  ├─ signInWithEmailAndPassword()
  ├─ createUserWithEmailAndPassword()
  └─ currentUser

FirebaseFirestore
  ├─ collection('profiles').doc().get()
  ├─ collection('profiles').doc().set()
  └─ Real-time listeners
```
